#Turtpress

Turtpress是一个轻量的静态Blog类站点生成器。使用Python编写，符合MVC设计模式，结构清晰。生成的站点可以免费托管在GitHub、Heroku这样的平台上，省去了host的费用。

#手册

去[turtpress.com](http://turtpress.com)查阅手册。
